# Basic Components

Here are some basic components constructed by styled-system with preset styles.

## Button

```js

<Button variant="primary">Save</Button>;
```

## FlexBox

```js
<Flex flexDirection="column" alignItems="center">
  <Box flex={1}>Box</Box>
  <Box flex={1}>Box</Box>
  <Box flex={1}>Box</Box>

```
